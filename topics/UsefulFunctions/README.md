# UsefulFunctions

Reusable utility functions for atomistic simulations — radial and angular
basis functions with finite support, smooth cutoffs, and efficient evaluation.

## Subfolders

| Folder | Topic |
|--------|-------|
| `AngularFunctions/` | Angular potentials for sp1, sp2, sp3 hybridization geometries using complex numbers and quaternions |
| `RadialFunctions/` | Radial basis functions with compact support, C2-continuous cutoff, and tunable width |
